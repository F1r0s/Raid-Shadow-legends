# Raid-Shadow-legends
Raid Shadow Legends Promo Codes Landing Page with Custom Content Locker Integration.

## Vercel deployment

1. Push this repository to GitHub.
2. In Vercel, click **Add New Project** and import this repository.
3. Keep framework as **Other** (static site) with no build command.
4. Deploy — Vercel will serve `index.html` as the landing page.

`vercel.json` is included with:
- clean URLs enabled
- long-term cache for image assets
- short edge cache + stale revalidation for HTML pages
